SOGEVIC is an adaptation of our extensible management software for stores.
It has been customized to implement a video store, not only taking care of the reservations, but also handling the registration of new clients and the stock of the store.

OKBoomer is a development team composed by junior developers.
We were given the task of continuing the development of a preexisting management software, but we decided to take it to another level.

Know more about them here

Working in groups and strengthen advance development on Java.
Reinforcing these concepts has been the primary goal of the project. In order to do so, we have made use of the git protocol along with Github as remote repository
We also implemented the Scrum framework using Taiga, which allows teams to easily work with an agile methodology.
On a more technical note we have made extensive use of interfaces, abstract clases, generics, collections, XML files which has result in a modular, scalable, easy to work and maintain software.
